## kubectl

### One command to create pod and service

```bash
kubectl run nginx --image=nginx --restart=Never --port=80 --expose
```

### Create temporary busybox pod

```bash
kubectl run busybox --image=busybox --restart=Never --rm -it -- /bin/sh
```

### Create manifests

<Tabs>
